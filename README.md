# 📝 Todo Full Stack App (Backend)

This is the backend part of a Full Stack Todo Application built using Node.js, Express, and MongoDB.

The project allows users to manage their tasks with authentication and secure API routes.

---

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Create, Read, Update, Delete (CRUD) Tasks
- Protected Routes (Authorization)
- Each user has their own tasks
- Secure password hashing using bcrypt
- RESTful API

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT (jsonwebtoken)
- bcryptjs
- dotenv
- cors
- nodemon
